# 🎨 前端

前端开发综合资源。

## 框架

- [Next.js](https://nextjs.org) — React 全栈框架
- [Nuxt](https://nuxt.com) — Vue 全栈框架
- [Astro](https://astro.build) — 内容驱动的静态站点框架
- [Rspress](https://rspress.dev) — 基于 Rspack 的静态站点生成器

## 状态管理

- [Zustand](https://github.com/pmndrs/zustand) — 轻量 React 状态管理
- [Pinia](https://pinia.vuejs.org) — Vue 状态管理
- [Jotai](https://jotai.org) — 原子化 React 状态管理

## 动画

- [GSAP](https://gsap.com) — 专业级 Web 动画库
- [Framer Motion](https://www.framer.com/motion) — React 动画库
- [Motion One](https://motion.dev) — 轻量动画引擎
- [Three.js](https://threejs.org) — 3D 图形库

## 测试

- [Vitest](https://vitest.dev) — 下一代测试框架
- [Playwright](https://playwright.dev) — 端到端测试
- [Testing Library](https://testing-library.com) — 组件测试
