# 🟢 Node.js

Node.js 生态资源。

## 运行时

- [Node.js](https://nodejs.org) — JavaScript 运行时
- [Bun](https://bun.sh) — 快速全功能运行时
- [Deno](https://deno.com) — 安全 JavaScript/TypeScript 运行时

## 框架

- [NestJS](https://nestjs.com) — 企业级 Node.js 框架
- [Express](https://expressjs.com) — Web 框架
- [Fastify](https://fastify.dev) — 高性能 Web 框架
- [Hono](https://hono.dev) — 轻量级 Web 框架

## ORM

- [Prisma](https://www.prisma.io) — 下一代 ORM
- [Drizzle](https://orm.drizzle.team) — 类型安全 ORM
- [TypeORM](https://typeorm.io) — TypeScript ORM

- [tsx](https://tsx.is) — TypeScript 执行器
- [PM2](https://pm2.keymetrics.io) — 进程管理器
- [pnpm](https://pnpm.io) — 快速磁盘高效的包管理器
