# 🛠 开发工具

日常开发必备工具。

## 编辑器

- [VS Code](https://code.visualstudio.com) — 代码编辑器
- [Cursor](https://cursor.sh) — AI 编辑器
- [Windsurf](https://codeium.com/windsurf) — 智能 IDE

## 终端

- [iTerm2](https://iterm2.com) — macOS 终端
- [Warp](https://www.warp.dev) — 现代化终端
- [Oh My Zsh](https://ohmyz.sh) — Zsh 配置框架

## 设计

- [Figma](https://www.figma.com) — 设计协作工具
- [MasterGo](https://mastergo.com) — 国产设计工具
- [Excalidraw](https://excalidraw.com) — 手绘风格白板

## API 工具

- [curl](https://curl.se) — 命令行 HTTP 工具
- [HTTPie](https://httpie.io) — 人性化 HTTP 客户端
- [Bruno](https://www.usebruno.com) — API 客户端（开源）
