# 🔧 工程化

前端工程化工具。

## 构建工具

- [Vite](https://cn.vite.dev) — 下一代前端工具链
- [Rspack](https://rspack.rs) — 基于 Rust 的高性能打包器
- [Rolldown](https://rolldown.rs) — Rust 版 Rollup
- [Webpack](https://webpack.js.org) — 经典打包工具

## 代码质量

- [ESLint](https://eslint.org) — 代码检查
- [Prettier](https://prettier.io) — 代码格式化
- [Biome](https://biomejs.dev) — 统一工具链
- [Commitlint](https://commitlint.js.org) — 提交信息规范

## 类型检查

- [TypeScript](https://www.typescriptlang.org) — JavaScript 超集
- [ts-reset](https://github.com/total-typescript/ts-reset) — 改进 TypeScript 类型

## CI/CD

- [GitHub Actions](https://github.com/features/actions) — CI/CD
- [Husky](https://typicode.github.io/husky) — Git Hooks
- [lint-staged](https://github.com/lint-staged/lint-staged) — 暂存区代码检查
