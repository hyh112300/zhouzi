# SSG 对比

静态站点生成器调研对比。

## 对比表

| 特性 | Docusaurus | VitePress | Next.js | Astro |
|------|-----------|-----------|---------|-------|
| 框架 | React | Vue | React | 多框架 |
| 文档功能 | ✅ 优秀 | ✅ 优秀 | 需手动搭建 | 需插件 |
| 博客 | ✅ 内置 | ✅ 内置 | ✅ 手动 | ✅ 内置 |
| 搜索 | ✅ 插件 | ✅ 内置 | 需集成 | 需集成 |
| i18n | ✅ 优秀 | ✅ 优秀 | ✅ 手动 | ✅ 集成 |
| 性能 | ⚡ 静态 | ⚡ 静态 | 🔥 SSR/SSG | ⚡ 静态 |

## 选择建议

- **文档站** → Docusaurus / VitePress
- **博客+文档** → Docusaurus（功能最全）
- **全栈应用** → Next.js
- **内容站点** → Astro（最佳性能）
