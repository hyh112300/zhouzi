# Markdown 方案

Markdown 解析与渲染方案选型。

## 规范

- [CommonMark](https://commonmark.org) — Markdown 标准规范
- [GFM (GitHub Flavored Markdown)](https://github.github.com/gfm) — GitHub 扩展语法

## 前端渲染

- [MDX](https://mdxjs.com) — React 组件 + Markdown（Docusaurus 使用）
- [remark](https://remark.js.org) — Markdown 解析器生态
- [marked](https://marked.js.org) — 快速 Markdown 编译器

## 编辑器

- [Tiptap](https://tiptap.dev) — 基于 ProseMirror 的富文本编辑器
- [Editor.js](https://editorjs.io) — 块级编辑器
- [Plate](https://platejs.org) — 基于 Slate 的 React 编辑器
