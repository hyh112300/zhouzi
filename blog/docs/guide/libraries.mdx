# 📚 常用库

开发中常用的第三方库推荐。

## 公共

- [TypeScript](https://www.typescriptlang.org) — 开发语言
- [es-toolkit](https://es-toolkit.dev) — 工具库（替代 lodash）
- [axios](https://axios-http.com) — 网络请求
- [clsx](https://github.com/lukeed/clsx) — 条件 className
- [dayjs](https://day.js.org) — 时间处理
- [zod](https://zod.dev) — 数据校验

## React

- [zustand](https://github.com/pmndrs/zustand) — 状态管理
- [TanStack Query](https://tanstack.com/query) — 异步状态管理
- [react-router](https://reactrouter.com) — 路由
- [Ant Design](https://ant.design) — 组件库

## 工程化

- [Vite](https://cn.vite.dev) — 构建工具
- [Vitest](https://vitest.dev) — 测试框架
- [ESLint](https://eslint.org) — 代码检查
- [Prettier](https://prettier.io) — 代码格式化
