# Docker

Docker 容器化配置与最佳实践。

## Dockerfile

```dockerfile
FROM node:20-alpine AS build
WORKDIR /app
COPY package*.json ./
RUN npm ci
COPY . .
RUN npm run build

FROM nginx:alpine
COPY --from=build /app/build /usr/share/nginx/html
COPY nginx.conf /etc/nginx/conf.d/default.conf
EXPOSE 80
CMD ["nginx", "-g", "daemon off;"]
```

## docker-compose

```yaml
version: '3'
services:
  app:
    build: .
    ports:
      - "80:80"
    restart: always
```
